イージング（easing）

サンプルコードの結果

サンプルコード

　　 float easing=0.05;

　　 void setup(){

　　 　size(480, 120);　　 　

　　 }

　　 void draw(){

　　 　float targetX=mouseX;

説明文

　イージングとは関数ではなくテクニックのことである。

　イージングを使うことで、線画より滑らかになるテクニックである。

　dist()関数では始点と終点の2点間の距離を求める関数であった。

　その算出された値に0.01や0.5といった数字を掛け算することによって、目的地へ少しずつ進んでいく線ができる。
